    退役心路

    这里是退役 OIer 们的精神角落。AFO 不是终点——那些在机房度过的日夜、一起刷题的队友、失利后的迷茫与重新出发，都是值得被记录的故事。

    这个板块是什么

    竞赛之外，OI 给我们留下的远不止算法。有人在这里找到了热爱的方向，有人学会了如何面对失败，也有人在这里交到了一辈子的朋友。

    这个板块收集退役 OIer 的真实故事——不是成功学，不是经验分享，而是属于我们的回忆与思考。

    投稿指南

    投稿方式

    1. Fork AFOIer 仓库
    2. 在 docs/stories/ 目录下创建新的 .md 文件
    3. 文件名使用英文小写 + 连字符，如 my-oi-memories.md
    4. 提交 PR，标题以 [故事投稿] 开头

    隐私说明

    • 支持匿名发布——如果你不想公开 GitHub 账号，可以请维护者代发（通过 Issue 提交）
    • 如果文中涉及他人的真实经历，请获得对方同意或做匿名化处理
    • 可以只写代号、省份简称，不要暴露具体学校和人名（除非当事人同意）

    写作建议

    • 不用写得很"正式"——这是回忆录，不是作文
    • 可以配图（机房照片、比赛照片等），放在 docs/assets/stories/ 目录
    • 字数不限，几百字到几千字都可以
    • 标题自定，文末可以加一句话总结或给后来者的话

    话题参考

    以下是一些投稿方向，但不受限于此：

    • AFO 之后的第一年：退役后的生活是怎样的？竞赛留下的习惯和思维还在影响你吗？
    • OI 教会我的事：除了算法，OI 还让你学到了什么？
    • 那些年一起刷题的队友：你们现在还有联系吗？各奔东西后还记得彼此吗？
    • 不是我想要的结果：竞赛失利的经历，以及之后的转折
    • 省队 / 国赛回忆：那些紧张的瞬间、意外的收获
    • 关于选择的思考：为什么选择 OI？退役后选择了什么方向？
    • 机房二三事：OI 生活中那些有趣、温暖或哭笑不得的小事
    • 教练的故事：那位让你印象深刻的教练
