# 职业发展

> 退役 OIer 散落在各种职业道路上——后端、前端、算法、量化、产品、创业、科研……这个板块收集真实的职业转型经历和行业观察，帮助后来者看到更多可能性。

## 这个板块是什么

竞赛训练带来的逻辑思维、快速学习能力、debug 般的排错直觉——这些"肌肉记忆"在职场中是真的有用的。但这个板块不做成功学包装，只收集真实的职业故事和方向参考。

**投稿方式**：Fork [AFOIer](https://github.com/AFOIer/AFOIer)，在 `docs/careers/` 下创建 `.md` 文件，提交 PR 标题以 `[职业分享]` 开头。支持匿名发布。

## 话题方向

- 从 OI 到 [某方向] 的转型经历
- 第一份工作怎么找、竞赛经历怎么写进简历
- 职场中那些 OI 训练带来的"肌肉记忆"
- 竞赛退役后，如何建立新的核心竞争力
- 工作中发现的"OI 没教你的事"
- 量化/科技/科研/创业等方向的选择思考
