# 泛 CS 类专业汇总

本部分包含:

- 以数据科学、智能科学等为培养方向的数学类强基专业
- 以微电子、集成电路、芯片设计等为培养方向的物理及应用物理类强基专业
- 培养方向偏向计算机、人工智能方向的信息与计算科学强基专业
- 偏向计算机、电子、自动化、机器人等方向的新工科及理工交叉强基专业
- 强基培养方式明确的包含计算机、自动化等专业的双学位、双专业项目

本部分**不**包含:

- 以纯数学为主的数学类强基专业
- 以理论物理、材料、核物理、天体物理等为主要方向的物理学类强基专业
- 各类不与强基培养方式直接关联，而需要自行在入校后申请的辅修、双学位、双专业项目

如未明确说明，高校排序均参照 [2026年强基计划报考指南](https://gaokao.chsi.com.cn/gkzt/jcxkzs) 中的顺序。受篇幅及时间限制，部分高校相应专业列表仅包含培养方案中的大类名称，如条件允许，后续将补充具体说明。

*斜体字*表示该专业较为偏向数学/物理等基础学科或基础学科学习难度显著高于同档次其他高校/专业（划分可能较为模糊，需自行辨别）。

⚠️ **信息来源说明**：以上专业列表来自各校强基计划招生简章和阳光高考平台。但请注意——**招生简章中写的培养方向，和你实际入学后能选到的方向之间可能有不小差距**。原因是：(1) 培养方案可能多年未更新；(2) 部分热门方向容量有限，进校后需要二次选拔；(3) 招生宣传中的"人工智能""智能科学"等词汇可能只是标签，实际课程体系仍以传统基础学科为主。**建议以目标院校在读学生的真实反馈为第一参考**，不要仅凭招生简章做决定。

## 如何理解这些专业名称

"信息与计算科学""智能科学与技术""数理基础科学"——对退役 OIer 来说，这些名称看起来都和 CS 沾边，但实际差异巨大。以下给出一个粗略的分类框架，帮助你判断一个专业到底"有多 CS"：

### A 类：本质上就是 CS 专业

典型特征：隶属计算机学院/人工智能学院，培养方案以 CS 核心课程为主，毕业可做纯软方向。

- **信息与计算科学**（计科院版本，如电子科技大学、华中科技大学）：课程体系接近 CS 本科，数据结构、操作系统、计网等核心课齐全
- **数学与应用数学**（人工智能学院版本）：数学和 CS 的混合，通常 CS 比例高于纯数学
- **智能科学与技术 / 人工智能**（独立专业）：新工科方向，以 AI/ML 为主线

### B 类：CS 占比可观但不是全部

- **信息与计算科学**（数学学院版本）：数学训练占大头，CS 课程可能集中在数值计算、算法等与数学交叉的部分。毕业做纯工程需要大量自学
- **数理基础科学**（集成电路方向）：偏硬件，和纯软差距较大

### C 类：以基础学科为主，CS 只是点缀

- **应用物理学**（电子科学方向）：主体是物理，CS 相关课程很有限。即使标签上写"集成电路""电子科学"，本科阶段大量时间仍在学四大力学
- **数学类**（非 CS 方向培养方案）：如果培养方案没有明确指向 CS/数据科学，大概率是按纯数学的路径走

> 以上分类是粗略的方向性判断，具体到每个学校、每个培养方案的差异需要逐个核实。**最可靠的判断方式是直接看培养方案中的课程表**——数一数 CS 核心课（数据结构、操作系统、计组、计网、数据库、编译原理等）占了多少学分，比看专业名称有用得多。

## OIer 选这类专业需要注意什么

1. **你的算法基础在这些专业中未必能直接变现**。强基专业通常大一/大二以数学和物理为基础课，前两年你可能感觉和 CS 没什么关系——这很正常，也是强基的"特色"。真正进入专业方向课后，你的算法功底才会开始发挥优势。

2. **双学位/辅修可能是更好的路径**。如果你被强基录取到一个数学/物理倾向的专业，而学校的双学位政策允许你拿 CS 学位，这有时候比直接读一个"伪 CS"专业更有性价比——数学学位 + CS 双学位的组合在保研/出国时往往比一个定位模糊的专业名吃香。

3. **小心"智能科学""数据科学"这类词**。它们出现在招生简章中通常是为了吸引生源，但落实到课程表上，有些学校只是把一两门机器学习/数据挖掘课加到传统数学/物理专业里，距离真正的 CS 训练还有不小距离。

4. **关注二次选拔政策**。部分高校在强基入学后有二次选拔机制（如分流到更偏工程的方向），这在招生简章中不一定写清楚，需要向在读学生了解。

5. **ACM 校队是重要补充**。如果你读的专业偏理论、coding 练习不够，加入学校 ACM 队可以弥补——既保持算法状态，也为保研/出国积累竞赛成绩。

## 就业与深造方向

各专业毕业去向的官方统计通常存在两个问题：(1) 滞后 2-3 年；(2) "深造率"可能包含各种类型的读研，不一定是你想去的方向。以下为定性方向，仅供参考：

- **偏 CS 的专业（A 类）**：和普通 CS 专业去向无明显差异——互联网、AI、量化、科研均有
- **偏数学/物理 + CS 混搭（B 类）**：深造比例高，去向覆盖 CS/数学/物理多个方向。如果研究生想转纯 CS，需要本科阶段自补工程类课程
- **偏基础学科（C 类）**：若想转 CS 就业，读研几乎是必经之路
