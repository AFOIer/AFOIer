# 高校就读体验

> 招生简章上的培养方案是官方叙事，在读学生的一手体验才接近真实。这个板块收集退役 OIer 在各大高校的就读感受——课程质量、科研氛围、竞赛背景的实际加成，以及那些"入学前应该知道"的事。

## 这个板块是什么

每篇就读体验按统一模板撰写，方便读者横向对比。你可以写自己的学校，也可以约同学一起写。

⚠️ 为了可信度，投稿时请注明**学校 + 专业 + 年级 + 竞赛背景**（如 "某C9 CS系 大二 退役省一"）。支持匿名发布，但请向维护者披露基本信息用于核实。

## 课程体验

### 专业核心课

- 课程质量如何？难度和深度是否合理？
- 有哪些"值得上"的课？有哪些"水课"？
- 课程对编程/数学基础的要求和 OI 训练重合度如何？

### 选修与通识

- 选课自由度大吗？
- 跨专业选课是否方便？

## 科研与竞赛

### 实验室 / 科研机会

- 本科生进组难度？
- 有哪些方向比较强？
- 导师的指导风格和人品（可模糊描述，不用点名）

### ACM 校队

- 有 ACM 传统吗？队内氛围如何？
- 训练机制和比赛机会

## OIer 的实际体验

### 竞赛背景的加成

- 课程中 OI 基础用得上的地方
- 保研/出国时竞赛成绩的实际作用
- 有没有因为竞赛经历获得的特殊机会？

### 不足与遗憾

- 入学前没想到的落差
- 如果重来一次会怎么选

## 投稿方式

1. Fork [AFOIer](https://github.com/AFOIer/AFOIer)
2. 在 `docs/experience/` 下创建 `.md` 文件，文件名用学校英文缩写（如 `pku-cs.md`、`zju-ai.md`）
3. 按模板填写，提交 PR，标题以 `[就读体验]` 开头

或通过 GitHub Issue 提交稿件，由维护者代为发布。
