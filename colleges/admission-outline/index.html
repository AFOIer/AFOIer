    综合评价招生

    综合评价招生是高考综合改革背景下，由高校依据高考成绩、高校考核成绩、学业水平测试成绩等多维度指标进行录取的招生方式。相比强基计划，综合评价招生覆盖的高校范围更广、专业选择更灵活，是退役 OIer 值得重点关注的升学通道。

    什么是综合评价招生

    综合评价招生（简称"综评"）突破了"唯分数论"的传统高考录取模式，采用 高考成绩 + 高校校测成绩 + 学业水平测试成绩 按比例折算的综合分进行录取。典型的计分比例为 \(6:3:1\)（高考 60%、校测 30%、学考 10%），但各高校可根据自身情况调整。

    与强基计划的区别

    维度 强基计划 综合评价
    招生定位 基础学科拔尖人才 综合素质优秀
    专业范围 数理化生文史哲等基础学科 覆盖更广，含热门应用类专业
    竞赛优惠 国二以上破格入围 竞赛奖项通过初审/校测体现
    录取批次 提前批之前 提前批或特殊批次
    院校层次 39 所双一流 A 类 院校层次更丰富

    对 OIer 的意义

    竞赛奖项在综合评价招生中虽不像强基破格那样有"直通"效果，但在以下环节有实际加分：

    • 初审环节：OI 省一/省二奖项是体现学科特长的有力材料，帮助通过初审筛选
    • 校测环节：综评校测普遍包含面试（部分高校有机试或笔试），OI 训练形成的逻辑思维和问题拆解能力在面试中有明显优势
    • 综合素质档案：竞赛经历是综合素质评价中的重要组成部分

    招生流程

    综合评价招生的一般时间线如下（以高三学年为参照）：

    高三上学期末 — 3 月：高校发布招生简章
3 月 — 5 月：考生网上报名，提交材料
5 月中下旬：高校完成初审，公布初审通过名单
高考结束后 — 出分前：高校组织校测（面试/笔试/机试）
高考出分后：确定入围名单，按综合分择优录取

    报名材料

    • 高中阶段成绩单
    • 学业水平测试成绩
    • 个人陈述 / 自荐信
    • 竞赛获奖证书（OI 省一/省二是关键加分项）
    • 推荐信（部分高校要求）
    • 综合素质评价档案

    校测形式

    综合评价的校测环节因校而异，常见形式包括：

    • 面试（最常见）：多对一或多对多面试，考察专业兴趣、逻辑思维、表达能力。OIer 在面试中被问及竞赛经历时，能够很好地展示自己的学习能力和专业热情。
    • 笔试（部分高校）：考察数学、物理等学科基础。OI 训练的数学功底在此处有用武之地。
    • 机试（少数高校）：需注意区分——少数高校（如部分 CS 强校）的机试直接考察编程能力，对 OIer 有利；但港中深、南科大等校的"机试"实为综合能力测试（数学、逻辑、英语等），并非编程上机考试，与 OI 能力无直接关系。具体以目标高校往年真题为准。
    • 小组讨论：考察团队协作与沟通能力。

    OIer 校测准备建议

    1. 准备好 1-2 分钟的竞赛经历介绍——重点不是你拿了什么奖，而是你学到了什么、解决了什么问题
    2. 如果目标高校有机试，先确认其性质——是编程上机考试还是综合能力测试，前者适度刷题保持手感即可（综评机试难度通常低于 OI 省选），后者需要按数学/逻辑方向备考
    3. 面试中可能被问到"为什么选择这个专业"，将 OI 经历和所选专业方向联系起来会很有说服力
    4. 关注目标高校往年综评校测真题（部分可在学校官网或经验帖中找到）

    代表性高校

    以下为近年来开展综合评价招生的部分高校（具体以当年招生简章为准）：

    面向多省份招生： - 南方科技大学 - 上海科技大学 - 上海纽约大学 - 昆山杜克大学 - 深圳北理莫斯科大学

    面向特定省份招生： - 浙江省三位一体：浙江大学、复旦大学、上海交通大学、以及省内绝大多数高校 - 上海市综合评价：复旦大学、上海交通大学、同济大学、华东师范大学、上海财经大学等 - 广东省综合评价：中山大学、华南理工大学、香港中文大学（深圳）、南方科技大学等 - 江苏省综合评价：南京大学、东南大学、南京师范大学等省属高校 - 山东省综合评价：山东大学、中国海洋大学、中国石油大学（华东）等

    各校具体招生政策每年可能调整，请以教育部阳光高考平台 (gaokao.chsi.com.cn) 及各高校招生简章为准。
